    <p>

      to create content structure that meets its purpose.
    </p>
  </div>
  <div class="feature">
    <h2>Modularity</h2>

      Content elements are modules which can be included on a per-need basis.
    </p>
  </div>
  <div class="feature">
    <h2>Interoperability</h2>

      applications in the ecosystem.
    </p>
